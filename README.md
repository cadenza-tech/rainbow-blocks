## Features

- Colorize code structure with nesting levels
- Customizable color palette

## Settings

| Setting | Default | Description |
| ------- | ------- | ----------- |
| `rainbowBlocks.colors` | `["#E07080", "#E0E070", "#70B0E0"]` | Array of colors (hex). Cycles when nesting exceeds length |
| `rainbowBlocks.debounceMs` | `100` | Update delay (0-1000ms) |

## Supported Languages

- Ruby (`.rb`)
- Elixir (`.ex`, `.exs`)
- Bash (`.sh`, `.bash`)
- Ada (`.adb`, `.ads`)
- AppleScript (`.applescript`, `.scpt`)
- COBOL (`.cob`, `.cbl`)
- Crystal (`.cr`)
- Erlang (`.erl`, `.hrl`)
- Fortran (`.f03`, `.f08`, `.f90`, `.f95`)
- Julia (`.jl`)
- Lua (`.lua`)
- Pascal/Delphi (`.pas`, `.pp`, `.dpr`)
- MATLAB (`.m`)
- Octave (`.m`)
- Verilog (`.v`, `.sv`)
- VHDL (`.vhd`, `.vhdl`)
